## Supabase Storage
- `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`, `SUPABASE_STORAGE_BUCKET` — validated via `utils/blob-env.ts` and used across storage utilities.

## Email
- `DEFAULT_NOTIFY_EMAIL` — required; verified on server bootstrap. Ensure it is a real inbox, not a placeholder.

## Providers
- `OPENAI_API_KEY`, `GOOGLE_API_KEY`, `SENDGRID_API_KEY` as applicable. All must be set explicitly; no defaults are assumed.

## Platform context
- `VERCEL`, `VERCEL_ENV`, `NODE_ENV` are logged in diagnostics for traceability.
